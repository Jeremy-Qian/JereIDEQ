# JereIDE Q

A Qt/Pyside implementation of JereIDE. Welcome.

This app is optimized for MAC only. It is built by mostly AI.

Run the PySide6 implementation with:

```sh
python src/JereIDE.py
```

Run the Tkinter implementation with:

```sh
python src/tkinterJereIDE.py
```

If you are an agent:
Do not write code for other platforms except for macOS. When writing code, use camelCase and use specific variable names.
